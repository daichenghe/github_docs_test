MA1181 通用数据协议¶

Start 1

Start 2

Frame type

Data content

Check 1

Check 2

Description:

  • Start: 每帧报文固定使用 2 字节: 0xBD 0x64.

  • Frame type: 1 字节功能码.

  • Data content: 59 字节.

  • Check: 2 字节 crc16 校验, 低字节在前, 校验范围为Start到最后一个Data content, 校验算法如下:

uint16_t CalculateCRC16(const uint8_t *puchMsg, uint32_t len)
{
    uint16_t wCRCin = 0xFFFFu;
    uint16_t wCPoly = 0x1021u;
    uint8_t wChar = 0u;
    uint32_t i;
    while (len--)
    {
        wChar = *(puchMsg++);
        wCRCin ^= (wChar << 8u);
        for(i = 0u; i < 8u; i++)
        if(wCRCin & 0x8000u)
        {
            wCRCin = (wCRCin <<   1u) ^ wCPoly;
        }
        else
        {
            wCRCin = wCRCin <<   1u;
        }
    }
    return (wCRCin);
}
